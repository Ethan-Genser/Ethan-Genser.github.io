My Projects

Steganographic Encoder

This is a Python script that allows the user to hide a secret image inside of a cover image. Later, the user can then use the program to reverse the encoding and regain an approximate copy of the original.

Maze Generator

This program procedurally generates mazes using a recursive backtracking algorithm. It is written in C++ and structured in an object-oriented manner.

Cause of Death Data Visualization Tool

This short python program takes specially formatted datasets from the Center for Disease Control and Prevention's website as input and outputs an xlsx file. The output file organizes the data into tables and creates a graphical chart to help visualize the information.

Conway's Game of Life

This is a model of John Conway's 'Game of Life'. The simulation is built in C# using Microsoft XNA as the framework and the Monogame Content Pipeline for a graphical interface.
